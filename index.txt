專案檔案結構說明：

- index.html: 應用程式的主要 HTML 結構。
- style.css: 所有的 CSS 樣式。
- app.js: JavaScript 邏輯，包含 Service Worker 註冊和表單計算。
- manifest.json: PWA 設定檔，定義 App 名稱、圖示等。
- sw.js: Service Worker 腳本，用於實現離線快取。
- /icons/ (資料夾): 存放您的應用程式圖示 (例如 icon-192.png, icon-512.png, apple-icon-180.png)。

部署注意事項：
- 確保所有檔案都已上傳到您的 GitHub 儲存庫。
- 在 GitHub Pages 設定中，確認部署來源是您的主分支。
- 由於 PWA 功能需要 HTTPS，使用 GitHub Pages 或 Netlify 是很好的選擇。
